# Feature Plan: Teams & Roles

## Objective
Support collaborative team workspaces with explicit role permissions.

## Scope
- Team creation and invitations
- Role model (Owner/Admin/Editor/Contributor/Viewer)
- Team library ownership and delegated access
- Role-aware actions for upload, edit, share, delete

## Planned detail expansion
- Team/member schemas and role inheritance
- Permission matrix by resource type
- Admin UX for membership management
- Security rule tests for role transitions
